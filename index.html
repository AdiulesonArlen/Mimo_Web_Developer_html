  • 1 Noções básicas de HTML
    • 1.1 Descobrindo HTML e Tags
      • 1.1.1 Tags HTML

        1.1.1 Tags HTML

        HTML é uma linguagem de marcação de computador que estrutura todas as páginas da web na internet.

        O código HTML informa aos navegadores da Web como exibir conteúdo com imagens, links e texto em uma página da Web.

        Em HTML, as coisas giram em torno de tags. As tags começam com <, seguidas por uma palavra-chave e terminam com >.

        Vamos criar nossa primeira tag <button> no arquivo abaixo!

        <button>

        <button> é uma tag de abertura. Ele informa ao navegador da Web que queremos começar um botão.

        A maioria das tags em HTML vem em um par, elas têm uma tag de abertura e uma tag de fechamento. O que você codificou anteriormente é a tag de abertura de um elemento botão.

        Vamos codificar a tag de fechamento do botão adicionando </button> após a tag do botão de abertura.

        <button></button>
      • 1.1.2 Textos e títulos

        Em HTML, as palavras-chave de algumas tags são óbvias, mas existem algumas tags que usam uma forma abreviada. Por exemplo, usamos <p> para um parágrafo.

        Codificar um parágrafo é muito semelhante a codificar um botão. Você precisa de uma tag de abertura, seguida de seu texto e uma tag de fechamento.

        <p>Aprender Desenvolvimento Web é Divertido!</p>

        A tag de cabeçalho enfatiza os textos, tornando-os maiores e mais ousados. <h1> é o maior e mais negrito e <h6> é o menor e menos negrito.

        <h1>Título Principal da Página</h1>

<h2>Título de tópicos</h2>

<h3>Título de Subtópicos</h3>
      • 1.1.3 Estrutura da página Web

        Um elemento é o que chamamos de tag de abertura, tag de fechamento e código intermediário.

        <button>Hello, World!</button>

        Para criar uma página da Web, você precisará de uma tag HTML de abertura e fechamento.

        <html>

</html>

        Os elementos html e body (corpo) formam a estrutura de uma página da web. Colocamos os elementos que queremos exibir dentro do corpo da página.

        <html>
	<body>

  </body>
</html>
    • 1.2 Estruturando texto com tags
      • 1.2.1 Quebra de linha, ênfase e importância

        Às vezes queremos formatar parágrafos com quebras de linha.

        Por exemplo, essa música ficaria muito melhor se Raindrops on rosesaparecesse em uma linha, Whiskers on kittensna próxima e assim por diante.

        <html>
	 <body>
		  <h1>My favorite things</h1>
		  <p>
		   Raindrops on roses <br>
		   Whiskers on kittens <br>
		   Bright copper kettles <br>
		   Warm woolen mittens
		  </p>
	 </body>
</html>

        <br> é uma tag vazia. Tags vazias não possuem tag de fechamento e nenhum conteúdo. Eles também são chamados de tags de fechamento automático.

        Há um elemento para dar ênfase ao texto, torna-o itálico . Chama-se em elemento. Adicione <em> e </em>.

        <html>
	 <body>
		  <h1>My favorite things</h1>
		  <p>
		   Raindrops on <em> roses </em><br>
		   Whiskers on kittens<br>
		   Bright copper kettles<br>
		   Warm woolen mittens
		  </p>
	 </body>
</html>

        Para definir textos como importantes, usamos as tags <strong>e . </strong>Torna o texto em negrito .

        <html>
	 <body>
		  <h1>My favorite things</h1>
		  <p>
		   Raindrops on <em>roses</em><br>
		   Whiskers on <strong>kittens</strong> 
		  </p>
	 </body>
</html>
      • 1.2.2 Cabeçalho, título e doctype

        O headelemento HTML, <head></head>, é um contêiner para informações sobre uma página da Web, como seu título. Ele vai antes da etiqueta do corpo.

        <html>
	 <head>
	
	 </head>
	 <body>
	 
	 </body>
</html>

        O título de uma página da web aparece na guia ou janela do navegador da web.

        Nas guias abaixo, há dois títulos de página da web: Minha página da web e Google .

        Para dar um título a uma página da Web, adicionamos <title>e </title>dentro das tags head.

        Observe que o titlenão é visível, mas fornece informações ao navegador.

        <html>
	 <head>
		 <title>My webpage</title>
	 </head>
		 <body>
			  <h1>My favorite things</h1>
			  <p>
			  Raindrops on <em>roses</em><br>
			  Whiskers on <strong>kittens</strong> 
			  </p>
		 </body>
</html>

        O doctype informa ao navegador da Web qual versão do HTML estamos usando. Sem ele, o navegador pode não exibir a página corretamente.

        Como estamos usando a versão mais recente, usamos <!doctype html>.

        <!doctype html>
<html>
	 <head>
		 <title>My webpage</title>
	 </head>
	 <body>
		  <h1>My favorite things</h1>
		  <p> Raindrops on <em>roses</em><br>
			Whiskers on <strong>kittens</strong></p>  
	 </body>
</html>

        O doctype se parece com uma doctype tag vazia com um html atributo vazio: <!doctype html>.

        Os atributos fornecem informações adicionais às tags. Você aprenderá mais sobre eles em capítulos posteriores.

    • 1.3 Botões de construção
      • 1.3.1 Botões

        Com botões, os programas alteram as páginas da Web com base no comportamento do usuário.

        Aprenderemos como escrever esse código mais tarde. Por enquanto, vamos nos concentrar no elemento button.

        Para criar um botão que os usuários possam clicar, precisamos das tags de abertura <button>e fechamento </button>.

        Podemos fazer com que os textos apareçam em um botão digitando-os entre as tags.

        <!doctype html>
<html>
 <body>
  <h1>Facebook</h1>
  <p>
		Log in to your account to get started
	</p>
  <button>Login</button>
 </body>
</html>

        Anteriormente, aprendemos como enfatizar ou tornar o texto importante. Também podemos usar essas tags para o texto dentro de um botão.

        <!doctype html>
<html>
 <body>
  <h3>Warning</h3>
  <p>
		This site contains humor that may cause uncontrollable laughter.
	</p>
  <button><strong>continue anyway</strong></button>
 </body>
</html>

        Os botões são exibidos um ao lado do outro, mesmo que estejam em linhas de código separadas.

        <!doctype html>
<html>
 <body>
  <button>First</button>
  <button>Second</button>
  <button>Third</button>
 </body>
</html>

        Se quisermos que os botões sejam empilhados uns sobre os outros, podemos usar a tag <br> para colocar cada botão em uma nova linha.

        <!doctype html>
<html>
 <body>
  <h3>Stack of Pancakes</h3>
  <p>
Log in or sign up to view the best photos of pancakes across the globe.
	</p>
  <button>Login</button>  
	<br>
  <button>Sign up</button>
 </body>
</html>
    • 1.4 Criando Links
      • 1.4.1 Links e atributos

        Vamos dar uma olhada em como os links funcionam para combinar páginas da Web em sites.

        Para começar a criar um link, adicionamos o texto entre as tags <a>e </a>. Isso ainda não destacará o texto.

        <a>significa tag "Âncora".

        <!doctype html>
<html>
 <body>  
	<a>Learn to Code</a>
 </body>
</html>

        Para vincular o texto a uma página da Web, adicionamos href=""um Uniform Resource Locator (URL). hrefé a abreviação de "referência de hipertexto".

        <!doctype html>
<html>
 <body>
  <a href="https://getmimo.com">Learn to Code</a>
 </body>
</html>

        hrefé um atributo . Todos os atributos têm duas coisas em comum: eles fornecem informações extras e vão dentro da tag de abertura.

        Os atributos são adicionados após o nome da tag e antes do >sinal de fechamento.

        Vinculamos a uma página da Web adicionando =após o atributo e um URL como um valor entre aspas.

        <!doctype html>
<html>
 <body>  
	<a href="https://getmimo.com">Best place to learn to code</a>
 </body>
</html>
